5. Data Security

We implement industry-standard security measures to protect your data:

  • Encryption in Transit: All data transmitted via HTTPS/TLS 1.2+
  • Encryption at Rest: Sensitive data encrypted in MongoDB Atlas
  • Password Security: Passwords hashed with bcrypt (never stored in plain text)
  • Access Controls: JWT-based authentication, role-based access control
  • Rate Limiting: Protection against brute-force attacks and DDoS
  • Content Moderation: Automated profanity filtering and spam detection
  • Regular Audits: Security reviews and vulnerability scans
  • Secure Infrastructure: Cloud hosting with SOC 2 compliance (Render.com, MongoDB Atlas)
  • Automated Backups: Regular data backups with encryption
  • Incident Response: Procedures to handle security breaches promptly

PeopleRate complies with Indian data protection laws and regulations:

We operate in accordance with the IT Act and its amendments, including:

  • Section 43A (Reasonable security practices)
  • Section 72A (Punishment for disclosure of personal information)
  • Section 79 (Intermediary liability protections)

10.2 Reasonable Security Practices (Rule 8, IT Rules 2011)

We maintain:

  • Comprehensive information security policies
  • Privacy policy (this document) published on our website
  • Security practices for data protection (see Section 5)
  • Procedures for handling data breaches

10.3 Digital Personal Data Protection Act (Future Compliance)

We are monitoring India's Digital Personal Data Protection Act (DPDPA) and will ensure full compliance once it comes into effect. This includes:

  • User rights for data access, correction, and deletion
  • Data breach notification within mandated timeframes
  • Appointment of Data Protection Officer (if required)

# PeopleRate - Render.com Deployment Guide

## üöÄ Quick Deployment Steps

### 1. Create Render.com Account
1. Go to https://render.com
2. Click "Get Started for Free"
3. Sign up with GitHub (recommended) or email

### 2. Push Code to GitHub
```bash
# Initialize git if not already done
git init
git add .
git commit -m "Initial commit - ready for deployment"

# Create new repo on GitHub, then:
git remote add origin https://github.com/YOUR_USERNAME/PeopleRate.git
git branch -M main
git push -u origin main
```

### 3. Deploy on Render.com

#### Option A: Using render.yaml (Automatic)
1. Click "New +" ‚Üí "Blueprint"
2. Connect your GitHub repository
3. Render will auto-detect `render.yaml`
4. Click "Apply" - deployment starts automatically!

#### Option B: Manual Setup
1. Click "New +" ‚Üí "Web Service"
2. Connect your GitHub repository
3. Configure:
   - **Name:** peopleRate
   - **Environment:** Python 3
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `uvicorn main:app --host 0.0.0.0 --port $PORT`
4. Add Environment Variables (see below)
5. Click "Create Web Service"

### 4. Environment Variables
Render will auto-generate `SECRET_KEY`. Add these manually if needed:

```
SECRET_KEY=<auto-generated>
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
ENVIRONMENT=production
CORS_ORIGINS=https://peopleRate.onrender.com
RATE_LIMIT_ENABLED=true
RATE_LIMIT_DEFAULT=100/minute
RATE_LIMIT_SEARCH=20/minute
RATE_LIMIT_REGISTER=5/hour
RATE_LIMIT_LOGIN=10/minute
```

### 5. Monitor Deployment
- Watch logs in Render dashboard
- First deployment takes 3-5 minutes
- Site will be live at: `https://peopleRate.onrender.com`

## üêõ Troubleshooting

### Deployment Failed
- Check logs in Render dashboard
- Verify `requirements.txt` has all dependencies
- Ensure Python version matches `runtime.txt`

### Site Won't Load
- Check if service is "Live" in dashboard
- Free tier sleeps - wait 60 seconds for wake-up
- Check logs for errors

### Database Issues
- In-memory mode works without MongoDB
- For persistence, add MongoDB Atlas connection
